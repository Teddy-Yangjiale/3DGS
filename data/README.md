# Data Directory

Put datasets here locally for testing. On the lab server, put the real datasets under:

```text
/data2/$USER/3DGS/data/
```

Recommended structure:

```text
data/
  lerf_mask/
    figurines/
    ramen/
    teatime/
  mipnerf360/
  custom/
    desk_objects/
```

This directory is ignored by Git except for this README.
